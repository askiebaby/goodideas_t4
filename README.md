
好想工作室切版練習（第四個樣版）
== 

## 使用技術與資源

- RWD 斷點: `576px`, `768px`, `992px`, `1200px`
- CSS Framework: Bootstrap 4
- Fontawesome
- Google Fonts
- CSS Sprites

## 介紹：

- 使用 `Bootstrap 4` 切版的練習，主要是練習對 `grid system` 及元件的熟悉度，運用 `card` 及 `navbar`元件。
- 切版方向：練習以手機裝置為主的切版，從 `小裝置寬度` 切到 `大裝置寬度`。
